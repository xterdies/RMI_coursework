# Graphic Materials

This directory contains the graphic materials prepared for the Regional Clustering Platform coursework.

Files:

- `01-bpmn-domain-processes.svg` - BPMN-style model of the domain processes
- `02-software-architecture.svg` - software architecture poster
- `03-database-model.svg` - database model schema
- `04-presentation-models.svg` - presentation/view model poster
- `05-core-business-logic-algorithm.svg` - flowchart of the main business logic

Format:

- Vector `SVG`
- A4 landscape layout
- Suitable for export to PDF or insertion into the report
